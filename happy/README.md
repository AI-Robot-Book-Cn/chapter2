# happy
## 概要
２章のサンプルプログラム 


## インストール
Chapter2の全パッケージは入手済みなのでパッケージをビルドするだけ．
- パッケージのビルド
  ```
  cd ~/airobot_ws
  colcon build --packages-select happy
  ```


## 実行  
端末で以下を実行
```
source install/setup.bash
ros2 run happy happy_node
```
